dt-money

# Instalation
1. **Clone the repository:** <br/>
`git clone https://github.com/lidianerfaria/dt-money.git`

2. **Install the dependencies:** <br/>
`yarn`

3. **Run the project:** <br/>
`yarn run dev`

4. **Run the API:** <br/>
`yarn dev:server`
